**📊 GLP-1 Weight Loss Drugs Master Dataset (2017–2026)**

**Overview :**
-- This project analyzes the GLP-1 Weight Loss Drugs Master Dataset (2017–2026) sourced from Kaggle.
-- The dataset contains adverse event reports, patient demographics, and reaction outcomes related to 
popular GLP-1 drugs such as Ozempic, Wegovy, Mounjaro, and Zepbound.


## Using Power BI, I built an interactive dashboard to explore drug safety trends, patient reactions, and demographic insights across multiple years and countries.

**📂 Dataset Information**

Source: Kaggle – GLP-1 Weight Loss Drugs Master Dataset 2017–2026
Time Period: 2017–2026

**Key Features:**

Patient demographics (age, gender, weight, country)
Reaction outcomes (Fatal, Not recovered, Recovered, Recovering, Unknown)
Safety report IDs by year
Drug-related adverse events


**🔎 Project Objectives**

Analyze adverse event trends over time.
Identify age group distribution of reactions.
Compare average patient age and weight across genders and countries.
Visualize reaction outcomes segmented by age groups.
Provide data-driven insights into drug safety and patient impact.


**📊 Dashboard Highlights**

The Power BI dashboard includes:

-- Average Patient Age by Country (bar chart)
-- Reaction Outcomes by Age Group (area chart)
-- Safety Reports by Year (2015–2025) (bar chart)
-- Reaction Count by Age Group (donut chart with % distribution)


**Key Metrics Cards:**

Average Age: Female – 58.07 | Male – 60.05

**🚀 Tools & Technologies**


Power BI – for interactive dashboard creation
Excel / CSV – dataset preprocessing
GitHub – portfolio hosting and version control




**📈 Key Insights**


- Adults (36–55) and Young (21–35) age groups account for the majority of reported reactions.
- Recovered outcomes dominate, but fatal and unknown cases highlight safety concerns.
- Safety reports have steadily increased from 2017 to 2025, reflecting growing usage and monitoring of GLP-1 drugs.
